# ⏰ Digital Clock

A modern and responsive **Digital Clock Web Application** built using **HTML, CSS, and JavaScript**, featuring a sleek glassmorphism design with real-time updates, date display, and AM/PM format.

---

## ✨ Features

* 🕒 Real-time clock (updates every second)
* 🌓 12-hour format with AM/PM indicator
* 📅 Displays full date (Day, Month, Year)
* 🎨 Glassmorphism UI with gradient background
* 💡 Smooth and minimal design
* 💻 Clean layout with proper spacing (no overlap)

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling (Flexbox, Glass UI, Gradients)
* **JavaScript (Vanilla)** – Dynamic time & date updates

---

## ⚙️ How It Works

* Uses JavaScript `Date()` object to fetch current time.
* Updates time every second using `setInterval()`.
* Converts 24-hour format into **12-hour format with AM/PM**.
* Displays formatted date using `toLocaleDateString()`.
